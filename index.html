This web app uses the free Hipolabs Universities API to fetch university data based on a country or an Indian state. When the user searches by country, the API filters results directly. When searching by state, the app fetches all Indian universities first and filters them client-side using the input keyword in the university name or `state-province` field.

Technologies used: HTML, CSS, JavaScript, Axios, and Hipolabs API. Designed with responsiveness and 3D animated UI effects for better user experience.
